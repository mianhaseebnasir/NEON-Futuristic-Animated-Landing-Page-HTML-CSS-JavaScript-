# 🚀 NEON – Futuristic Landing Page

A visually stunning and modern landing page designed with a futuristic neon theme. This project demonstrates clean UI/UX principles, smooth animations, and responsive design using pure HTML, CSS, and JavaScript.

---

## 📌 Overview

NEON is a high-quality landing page template built for developers, startups, and businesses who want a modern digital presence. It combines aesthetic design with performance and usability.

---

## ✨ Features

- ⚡ Smooth animations and transitions
- 🎨 Modern neon-inspired UI design
- 📱 Fully responsive layout (mobile-friendly)
- 🚀 Fast and optimized performance
- 🔒 Clean and structured codebase
- 💡 Interactive UI elements (buttons, counters, forms)
- 🌟 Section-based layout (Hero, Features, About, Contact)

---

## 🧱 Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
